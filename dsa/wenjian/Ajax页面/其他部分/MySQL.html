• 创建数据库: CREATE DATABASE 数据库名【create database】
  • 删除数据库: DROP DATABASE 库名【drop database】
  • 查看所有数据库: show databases【SHOW DATABASES】
  • 选中数据库: use 库名【USE】
  • 查看当前数据库中的表: show tables【SHOW TABLES】
  • 创建表: CREATE TABLE 表名(字符1 类型(长度),字符2 类型(长度)...)【create table】
  • 删除表: DROP TABLE 表名【drop table】
  • 修改表名 alter table 旧表名 rename 新表名
  • 查看表的创建语句: SHOW CREATE TABL 表名\G【show create table】
  • 查看表的详细信息/结构: DESC 表名【desc】
  • 指定表引擎和字符集(未测试): 指定引擎：ENGINE = lnnoDB (i还是l？) 指定表默认字符集：DEFAULT CHARSET = utf8
  • 修改表的字段类型: alter table 表名 modify 字段名 字段类型(长度) 示例：alter table biao1 modify hang1 varchar(32);
  • 增加表的字段: alter table 表名 add column 字段名 字段类型(长度) 示例：alter table biao1 add column hang2 varchar(32);
  • 增加表的字段时使其在另一个字段后面: alter table 表名 add 字段名 字段类型(长度) after 另一个字段 示例：alter table biao1 add hang3 int(32) after hang1
  • 增加表的字段时使其位置处于第一个:
  • 删除表字段 alter table 表名 drop column 字段名 示例：alter table biao1 drop column hang3
  • 表字段改名/表字段改名（改类型） alter table 表名 change 原字段名 新字段名 字段类型（长度） 示例：alter table biao1 change hang2 hang3 varchar(60)
  • 修改表字段排列顺序
